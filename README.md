# Bargelo_bot

Bargelo_bot is a telegram-bot, mobile-ready, offline-storage, Nodejs and powered HTML5.

# To start!

  - run docker-compose up -d
  - insert your bot key inside the index.js and try it!

## Installation

### Requirements
 - nodejs
 - npm
 - docker
 - docker-compose
```sh
$ cd bargelo_bot
$ docker-compose up -d
$ npm install --save
$ npm start
```
